# Practical 4 - Methods, Objects, Variables, and Constructors

This folder contains beginner-friendly Java solutions for all 11 questions in **CP 215 Practical 4**.

## Folder Structure

- `QuestionOne` to `QuestionEleven` each contain the solution for one question.
- Some questions use two files because the question asks for separate classes.
  - `QuestionSeven` uses `Student.java` and `QuestionSeven.java`.
  - `QuestionNine` uses `ProgramClass.java` and `QuestionNine.java`.
  - `QuestionTen` uses `Employee.java` and `QuestionTen.java`.
  - `QuestionEleven` uses `Invoice.java` and `QuestionEleven.java`.

## How to Compile and Run

For single-file questions:

```bash
cd QuestionOne
javac QuestionOne.java
java QuestionOne
```

For multi-file questions (example QuestionTen):

```bash
cd QuestionTen
javac *.java
java QuestionTen
```

## Notes

- Code is written in a simple beginner style.
- Inline comments are included to explain key code blocks.
