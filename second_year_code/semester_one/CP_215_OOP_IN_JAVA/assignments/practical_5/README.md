# Practical 5 - Methods, Objects, Variables, and Constructors

This folder contains beginner-friendly Java solutions for all 5 questions in **CP 215 Practical 5**.

## Folder Structure

- `QuestionOne` - `Account` class and basic testing with two objects.
- `QuestionTwo` - `withdraw` method added with balance validation.
- `QuestionThree` - static variable `count` and static method `getCount`.
- `QuestionFour` - `Rectangle` class with validation, area and perimeter.
- `QuestionFive` - `SavingsAccount` class with static interest rate.

## How to Compile and Run

For QuestionOne:

```bash
cd QuestionOne
javac *.java
java QuestionOne
```

For QuestionTwo to QuestionFive:

```bash
cd QuestionTwo
javac QuestionTwo.java
java QuestionTwo
```

Replace `QuestionTwo` with `QuestionThree`, `QuestionFour`, or `QuestionFive`.

## Notes

- Code is intentionally simple for beginners.
- Inline comments are included to explain each important block.
