# Practical 8 (Lab 8) - Exception Handling

This folder contains beginner-friendly solutions for all 7 questions in **CP 215 Lab 8**.

## Folder Structure

- `QuestionOne/QuestionOne.java` - Catch subclass exceptions using superclass catch block.
- `QuestionTwo/QuestionTwo.java` - Catch different exceptions using `catch (Exception e)`.
- `QuestionThree/QuestionThree.java` - Demonstrate why catch block order matters.
- `QuestionFour/QuestionFour.java` - Constructor failure and exception handling.
- `QuestionFive/QuestionFive.java` - Rethrowing exceptions.
- `QuestionSix/QuestionSix.java` - Catching exceptions in outer scope.
- `QuestionSeven/QuestionSeven.java` - Exception propagation example.

## How to Compile and Run

Use one question folder at a time.

```bash
cd QuestionOne
javac QuestionOne.java
java QuestionOne
```

Replace `QuestionOne` with `QuestionTwo`, `QuestionThree`, and so on.

## Notes

- Code is intentionally simple for beginners.
- Inline comments are included to explain the key blocks.
